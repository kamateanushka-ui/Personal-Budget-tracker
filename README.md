# 💰 Personal Budget Tracker

A simple web-based application to track daily expenses and manage personal finances.

## 🚀 Features
- Add and track expenses
- Categorize transactions
- Store data using SQLite database
- Simple and user-friendly interface

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: SQLite

## 📂 Project Structure
- /backend → server + database
- budget_tracker.html → frontend UI

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open:
   budget_tracker.html
